# flex-bison-compiler
Compiler construction using Flex and Bison

## Setup Instructions
Clone this repository. You will need `flex`, `bison`, and `gcc` installed on your machine.

#### To build:
`$ ./build.sh`
<br>
*compiled*: `compiler`, `grammar.tab.c`, `grammar.tab.h`, `lexer.yy.c`

#### To clear:
`$ ./clear.sh`
